前言

小鬥士 雄霸傳是由Itari改檔製作的LF2改版，是前作小鬥士 次元衝突的重製版。
這改版的目的是盡量減少和原版的平衡性偏差，
同時提供新系統，角色，遊玩感受和故事。

這改版增加了

  • 10名鬥士級別的新人物
  • 5名新/強化頭目
  • 全新的闖關模式
  • 第二個生存模式
  • 超過35個新背景
  • 以及橙色介面

v2.1更新內容

錯誤修正
  • GodLouis: 嘗試使用已移除的自動反攻時，不會再跳進不存在影格
  • GodBat: 在第二階段死亡時不會再卡死生存模式
  • Rudolf: 變身成Winnie，使用解放真身途中變回Rudolf後，不會再出現圖像錯誤
招式表修正
  • 重製讀我檔，提升可讀程度
  • 把所有持續型招式的MP消耗改為以秒計算
  • GodHongBa: 修正中文版整頁缺失，英文版中萬劍閃的MP消耗錯誤
  • GodBandit: 修正補上技的鐮刀攻擊MP消耗錯誤
  • Dennis: 修正百烈腳的派生錯誤
  • Winnie: 修正字體難以閱讀的問題
  • Woody: 偷襲拳收納在轉移中
  • Dante: 連環箭(空中)更名為降雨箭
招式修改
  • GodHongBa, GodBandit: 改良了自動反攻的視覺效果和音效
平衡修改
  • 單向無限防禦: MP不足時將進入新的弱點狀態，維持1.5秒，可被接近的敵人捕捉
  • 單向無限防禦，單向防禦: 現在拿著重物時也可格擋
  • GodHongBa, HongBa: 縮小了劍十字的偵測範圍，調整了劍十字預備時的被攻擊判定
  • Freeze: 冰抓殺MP消耗由40%上升至70%

闖關模式說明

任務系統
每當進入新的關卡，在闖關模式的左上方都會顯示通關條件

對話系統
玩家可選擇接收或是跳過大部分對話
每個小節的對話長達三秒
另外部分對話需要特定條件才可發動

條件分歧系統
在每一關都有條件分歧。
要解鎖隱藏內容，就必須以各自的條件通關。
另外，根據玩家選擇，遊戲共有九個結局

對話觸發方塊
在對話觸發方塊進行基本動作：攻擊（Ａ）或防禦（Ｄ）可以觸發對話。
方塊上有ignore（無視）的選項的話，玩家可以放置它，在五秒後將自動消失

兵器系統
—在闖關模式可以使用各式兵器，
不單有手動和自動之分，玩家也需要學習破壞敵方的兵器。

生存模式
本改版有兩個生存模式，難度大致相同。
新生存模式：和原版生存模式有著一樣的敵人分佈，但是部分鬥士和頭目是改版新增人物。
所有本改版的人物都會出現。
舊生存模式：除了敵人會使用新招式和受到少許調節外，和原版生存模式完全一樣。
不會出現改版新增人物。

生存模式選擇方法

在生存模式開始時五秒之內，可以在對話觸發方塊進行基本動作以選擇不同的生存模式。
如果不選擇會自動默認選擇新生存模式。

派生: 本招式中按某按鍵可以轉換其他招式

MP消耗:
單個數字: 每次發動MP消耗固定
A + B: 每次發動MP消耗根據個別數值
A...: 每次重覆發動MP消耗固定
A/秒: 持續發動，消耗速度以秒計算
