前言

LF2: Story of HongBa, remake of LF2 Hetero Clash is a LF2 modification made by Itari.
This modification is aimed to minimize balance and gameplay change from original LF2
And provide new systems, characters and a complete and new story.

This version added:

  • 10 hero-class characters
  • 5 new or/and reinforced bosses
  • New stage mode
  • Second survival mode
  • More than 35 new backgrounds
  • And orange graphical interface

Stage mode

Mission system
Whenever a new mission begins, mission criteria is shown on top left corner

Talk system
Player can choose to initiate or skip talks
Each sub-section of talk lasts 3 seconds
Some talk can only be triggered in certain conditions

Branched story system
In every stage there is branched story
To unlock hidden content, you must clear the stages under certain conditions
Under selections of players, one of nine different endings can be achieved

Talk trigger bubble
On these bubbles, you can perform basic actions: Attack or Defend to start talking
If the bubble has the Ignore option, it will automatically disappear after 5 seconds

Arsenal system
In stage mode, you can use a variety of artillery
Apart from manual and automatic ones, players will also learn destroying them

Survival Mode
This version has two survival modes which difficulties are roughly equal.
New survival mode: Enemy mapping is identical to survival mode in original LF2, but some heroes and bosses are exclusive characters to this version.
Every character in this version will appear.
Old survival mode：Identical to survival mode in original LF2, no character exclusive to this version will appear,
However characters will use new moves and are slightly adjusted

Choosing your survival mode

After starting survival mode, you have 5 seconds to move to Talk trigger bubble and perform basic action at it.
If ignored, new survival mode is chosen by default

Map

You may enter any keyword to search, for example Duel or Short

Map type:
Duel: Small maps with little obstruction
Battle, Stage, Survival: Large maps
Testing: Small maps convenient to be recorded for clear scene
Original: Also included in original LF2
Newest: Newest maps, listed after Stone Age

Specific effect:
None: No effect
Harmless scenery: Aesthetic effect that poses little distraction to battle
Object summon: Changes drop rate of common objects
Special object: Drops powerful map-specific object
Area effect: Causes powerful effect that may turn tide of battle

The last 3 maps are not included in random selection

Move list explanation

Key Set:
Defend D; Jump J; Attack A; Up ↑; Down ↓; Forward →;
Wait + ; Repeat (X...); Added move (X)
↳ means this move can only be cast by chaining from other moves

Chained move: Pressing key in a move can chain to another move

MP cost:
Single number: Cost is fixed for each cast
A + B: Each cast may consume different amount of MP
A...: Cost is fixed for repeated cast
A/second: Continued move which depletes MP in fixed rate

GodBat

Special Boss
This boss is divided in 3 phases

GodBat starts from Phase 2 if appearing as enemy in stage and survival

Phase 1:
After starting the fight, by pressing A he can use Ambush. Afterwards he gains approximately 16 points of armor
Red bats surrounding him can nullify projectiles but may obstruct Eye laser
Each killed bat decreases 1 point of armor

When Armor is active he is invincible, but he cannot jump or use weapon
